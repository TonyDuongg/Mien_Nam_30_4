# 🇻🇳 Việt Nam Tiến Lên - Hành Trình Thống Nhất

Một mini-game tương tác trên nền tảng web được xây dựng nhằm kỷ niệm ngày Giải phóng miền Nam thống nhất đất nước (30/4) và Quốc tế Lao động (1/5). Người chơi sẽ hóa thân thành người lái xe tăng, vượt qua các chặng đường lịch sử từ Bắc vào Nam bằng cách trả lời các câu hỏi về lịch sử Việt Nam để tiến vào Dinh Độc Lập.

## Các tính năng nổi bật mà mình đã làm:
* **Hệ thống Quiz**: Trả lời các câu hỏi trắc nghiệm có giới hạn thời gian để giúp xe tăng tiến lên các chặng tiếp theo.
* **Bản đồ hành trình**: Giao diện trực quan mô phỏng đường đi từ Hà Nội đến Sài Gòn với các animation mượt mà cho xe tăng và các cột mốc.
* **Vòng quay may mắn (Lucky Wheel)**: Tích hợp vòng quay phần thưởng ở cuối game.
* **Âm nhạc tích hợp**: Tích hợp nhạc nền tự động thông qua YouTube Iframe API với nút bật/tắt tiện dụng.
* **Giao diện Responsive**: Thiết kế CSS thuần, tương thích hoàn toàn với các thiết bị di động và màn hình nhỏ.

## Dự án được xây dựng hoàn toàn bằng các công nghệ nền tảng (Vanilla), không sử dụng thư viện hay framework bên ngoài để tối ưu hóa hiệu suất và thể hiện sự nắm vững các kiến thức cốt lõi:
* **HTML5:** Cấu trúc layout, tích hợp `<canvas>` và `<iframe>`.
* **CSS3:** Flexbox, CSS Grid, Custom Properties (Variables), CSS Animations & Keyframes (`@keyframes`).
* **Vanilla JavaScript (ES6+):** Xử lý DOM, Game Loop (`requestAnimationFrame`), quản lý State, hướng đối tượng (OOP) cơ bản cho hệ thống Particles/Fireworks.
